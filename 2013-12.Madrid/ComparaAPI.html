MethodLinkSpeciesSet

The MethodLinkSpeciesSet is a central component in the Compara database, it stores information connecting the various analyses (method) with a set of species (species_set_obj)

1. Print the name of the MethodLinkSpeciesSet for the pig-cow LASTZ-NET alignments.

2. Print the list of all MethodLinkSpeciesSet entries stored in the database

3. Print a list of the species and their internal Ids (dbIDs) for the 12 eutherian mammal EPO alignments

HINT: The species_set_name for the eutherian mammals is "mammals"
